# 六年级上册数学互动教学方案

这是基于北师大版六年级上册数学教材整理的 5 周强化教学方案，适合基础薄弱的孩子集中补弱。

## 文件说明

- `index.html`：入口页，自动跳转到交互教学页面。
- `六上北师大版数学互动教学.html`：可交互教学工作台，包含 35 次逐课学习路线、每课 4 步学法、对应必背公式、过关标准、训练清单和陪学记录。
- `六上北师大版数学教学方案.md`：可打印或二次编辑的文字版方案。
- `assets/`：交互页面引用的教材代表页图片。

## 使用方式

直接用浏览器打开 `index.html` 或 `六上北师大版数学互动教学.html`。

页面中的勾选进度和记录会保存在当前浏览器本地。
